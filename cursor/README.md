# Cursor Review Plugin

Cursor Agent CLI (Composer 2) を使用して、実装計画・設計・コードをレビューするプラグイン。

## インストール

```bash
/plugin install cursor@relu-marketplace
```

## 使い方

```
/cursor
```

または自然な言葉で:
- 「cursorと相談」
- 「cursorに聞いて」
- 「caと相談」
- 「caでレビュー」
- 「Composerに聞いて」

## 機能

- **実装計画のレビュー**: 計画の整合性・抜け漏れを確認
- **設計のレビュー**: アーキテクチャ・設計判断の妥当性を確認
- **コードレビュー**: コード品質・バグ・改善点を確認
- **モード**: `ask`（レビュー・分析・質問すべてに使用）
- **自律的な議論**: Cursorの回答に疑問があれば、自動的に議論を開始

## 前提条件

- [Cursor CLI](https://www.cursor.com/) がインストールされていること
