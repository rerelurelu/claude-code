# Codex Review Plugin

Codex CLI (OpenAI) を使用して、実装計画・設計・コードをレビューするプラグイン。

## インストール

```bash
/plugin install codex@relu-marketplace
```

## 使い方

```
/codex
```

または自然な言葉で:
- 「codexと相談」
- 「codexに聞いて」
- 「コードレビュー」
- 「レビューして」

## 機能

- **実装計画のレビュー**: 計画の整合性・抜け漏れを確認
- **設計のレビュー**: アーキテクチャ・設計判断の妥当性を確認
- **コードレビュー**: コード品質・バグ・改善点を確認
- **自律的な議論**: Codexの回答に疑問があれば、自動的に議論を開始

## 前提条件

- [Codex CLI](https://github.com/openai/codex) がインストールされていること
